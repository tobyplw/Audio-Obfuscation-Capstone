# Audio Obfuscation Capstone

## Dependencies
-   dnspython 
-   pymongo 
-   certifi 
-   customtkinter
-   tk 
-   bcrypt
-   public-ip
-   deepgram-sdk
-   datetime
-   PIL
-   pylibsrtp
-   pyaudio
-   random
-   numpy
-   stun
-   googletrans
-   pyperclip
-   scipy
-   pedalboard


Run the following to install all dependancies:
```bash
pip install -r requirements.txt
```

## Usage 

Simply run:
```bash
py main.py 
```
or (depending on interpreter)
```bash
python main.py 
```
