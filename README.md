# Audio Obfuscation Capstone

## Dependencies
-   dnspython 
-   pymongo 
-   certifi 
-   customtkinter 
-   bcrypt
-   googletrans

Run the following to install all dependancies:
```bash
pip install -r requirements.txt
```

## Usage 

Simply run:
```bash
py main.py 
```
or (depending on interpreter)
```bash
python main.py 
```
